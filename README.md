# SEG File Comparison Tool / SEG文件比较工具

**SEG File Comparison Tool** is a professional-grade file comparison utility specifically designed for analyzing and comparing SEG file labels. It provides comprehensive comparison capabilities with detailed error analysis and intelligent file categorization.  
**SEG文件比较工具**是一款专业级的文件比较工具，专门用于分析和比较SEG文件标签。该工具提供全面的比较功能，包含详细的错误分析和智能文件分类。

## Usage / 使用说明

### Basic Workflow / 基本工作流程

1. **Set Directories** / 设置目录
   - Select reference directory containing ground truth SEG files / 选择包含真实标签SEG文件的参考目录
   - Select prediction directory containing predicted SEG files / 选择包含预测SEG文件的预测目录

2. **Start Comparison** / 开始比较
   - Click "Start Comparison" button to begin analysis / 点击"开始比较"按钮开始分析
   - Monitor progress in real-time / 实时监控进度

3. **View Results** / 查看结果
   - Check summary statistics in the results tab / 在结果标签页查看汇总统计
   - Examine detailed file comparison results / 检查详细的文件比较结果
   - Analyze files by error categories / 按错误类别分析文件

4. **File Operations** / 文件操作
   - Select error category for file operations / 选择错误类别进行文件操作
   - Choose source files (reference, prediction, or both) / 选择源文件（参考、预测或两者）
   - Specify target directory and copy files / 指定目标目录并复制文件
